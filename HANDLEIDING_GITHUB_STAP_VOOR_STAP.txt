Pokémon TCG Checklist online zetten via GitHub Pages
====================================================

Voorbereiding
-------------
1. Download het GitHub-pakket van ChatGPT.
2. Pak de zip uit op je computer.
3. Laat de mapstructuur intact. Vooral deze map moet blijven bestaan:
   .github/workflows/update-cardmarket-prices.yml

Stap 1 - Nieuwe repository maken
--------------------------------
1. Ga naar GitHub en meld je aan.
2. Klik rechtsboven op het plus-icoon (+).
3. Kies New repository.
4. Repository name: pokemon-tcg-checklist
5. Zet de repository op Public.
6. Zet Add a README file uit als dat aangevinkt staat.
7. Klik op Create repository.

Stap 2 - Bestanden uploaden
---------------------------
1. Open je nieuwe repository.
2. Klik op uploading an existing file.
3. Sleep alle bestanden en mappen uit de uitgepakte map naar GitHub.
   Dus niet de zip zelf uploaden, maar de inhoud van de map.
4. Controleer dat ook deze map geüpload is:
   .github/workflows/
5. Onderaan bij commit message mag je zetten:
   Eerste versie Pokémon TCG Checklist
6. Klik op Commit changes.

Stap 3 - GitHub Actions schrijfrechten geven
--------------------------------------------
1. Ga in je repository naar Settings.
2. Klik links op Actions.
3. Klik op General.
4. Scroll naar Workflow permissions.
5. Kies Read and write permissions.
6. Klik op Save.

Dit is nodig omdat de dagelijkse prijsupdate prices.json, price_history.json,
cardmarket_match_report.csv en cardmarket_setcodes.csv automatisch moet kunnen bewaren.

Stap 4 - GitHub Pages aanzetten
-------------------------------
1. Ga in je repository naar Settings.
2. Klik links op Pages.
3. Bij Build and deployment kies je:
   Source: Deploy from a branch
4. Bij Branch kies je:
   main
5. Bij map kies je:
   /root
6. Klik op Save.

Na enkele minuten toont GitHub bovenaan de Pages-link.
Die ziet er ongeveer zo uit:
https://jouwgebruikersnaam.github.io/pokemon-tcg-checklist/

Stap 5 - Eerste prijsupdate manueel starten
-------------------------------------------
1. Ga naar het tabblad Actions.
2. Klik links op Update Cardmarket prijzen.
3. Klik op Run workflow.
4. Laat Branch op main staan.
5. Klik opnieuw op Run workflow.
6. Wacht tot het groene vinkje verschijnt.

Daarna zijn deze bestanden automatisch gevuld:
- prices.json
- price_history.json
- cardmarket_match_report.csv
- cardmarket_setcodes.csv

Stap 6 - App testen op computer
-------------------------------
1. Open de GitHub Pages-link.
2. De app opent in je browser.
3. Klik eventueel op Prijzen ophalen.
4. Controleer of bij Cardmarket prijzen staat hoeveel prijzen zijn ingeladen.

Stap 7 - App op iPhone installeren
----------------------------------
1. Open de GitHub Pages-link in Safari op je iPhone.
2. Tik onderaan op het deel-icoon.
3. Kies Zet op beginscherm.
4. Geef de app eventueel de naam Pokémon TCG.
5. Tik op Voeg toe.

Vanaf nu staat de app als icoon op je beginscherm.

Belangrijke opmerking over je eigen collectie
---------------------------------------------
Je vinkjes, notities, conditie en varianten worden lokaal op je toestel bewaard.
Ze worden dus niet automatisch tussen iPhone en computer gesynchroniseerd.
Maak regelmatig een backup via de knop Backup opslaan in de app.

Dagelijkse update
-----------------
GitHub Actions haalt dagelijks de publieke Cardmarket-bestanden op en maakt nieuwe prijsbestanden.
Je hoeft hiervoor niets meer op Windows te starten.

Als prijzen niet verschijnen
----------------------------
Controleer dit:
1. Staat de GitHub Pages-link juist?
2. Is de eerste workflow al gelopen met een groen vinkje?
3. Staat prices.json in de repository?
4. Staat GitHub Actions op Read and write permissions?
5. Open de app en klik op Prijzen ophalen.
